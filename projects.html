  • Xiangqi Chess Game

    The topic of this project was a special version of chess called Xiangqi. We had the following tasks:
    Server-> Implement the specific rules of the Xiangqi within the game server framework. The implementation was done in an object-oriented manner using Java.
    Quality Assurance-> To ensure a correct game progression, the functionality implemented must be extensively tested using JUnit. Additionally, the code quality should be measured and ensured using certain metrics.
    Bot-> Develop a bot in Haskell, so that a human can play against a computer player. The bot should calculate all possible moves for given, valid game states and select one.
